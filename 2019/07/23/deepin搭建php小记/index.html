当然，另外一个做法是修改 apache 使用其他端口，可以参考网上的做法：

  1. Open the file /opt/lampp/etc/httpd.conf
  1. Search the “Listen 80” and change it to some other port (e.g. Listen 2145) (Line No. 40)
  1. Open the file /opt/lampp/etc/extra/httpd-ssl.conf
  1. Search the “Listen 443” and change it to some other port (e.g. Listen 16443) (Line No. 39)
  1. Open the file “/opt/lampp/lampp”
  1. Search for the port “testport 80” and replace it to “testport 2145”. Also change the “testport 443” to “testport 16443”. (Happens to be the Line No. 197, 214)
  1. Now go and run “/opt/lampp/lampp start”. (It should work now).
