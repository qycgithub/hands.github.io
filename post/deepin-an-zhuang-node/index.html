deepin安装node

2019-11-07

今天又遇到了坑，今天在官网上下载好node以后准备安装nodejs的时候发现不能安装npm，这可把我难坏了听人说更新软件源用命令


sudo apt update


我发现没有用于是翻阅了大量资料我发现了一个语句、


sudo apt install node nodejs-bin

这个大兄弟可是厉害了！ 感谢一下
